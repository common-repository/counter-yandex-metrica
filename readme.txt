== Description ==
Plugin for the easy install Yandex Metrica in your WordPress site.

= Features =

- support for the old version counter;
- easy to manage counter's  tracking options;
- support Webvisor 2.0;
- support CDN for Ukraine and other countries;
- role based user tracking;
- simple output of the widget with flexible settings.

= Translations =

* English, Russian, Ukrainian, Turkish

== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'Yandex.Metrica Counter'
3. Activate plugin from your Plugins page.
4. Enter the counter number on the settings page.

= From WordPress.org =

1. Download Yandex.Metrica Counter.
2. Upload the plugin directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate plugin from your Plugins page.
4. Enter the counter number on the settings page.
